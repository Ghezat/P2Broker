Tx Hash es el hash de la transacción. También se conoce como ID de transacción (TxID). Consta de caracteres alfanuméricos y representa el número de identificación especificado para la transacción en una blockchain. Cada vez que se ejecuta una transacción queda registrado en un hash TxID, de tal manera que puedes verificar en todo momento todos los datos de dicha operación. Aquí hay un ejemplo de un ID de transacción de Bitcoin.
